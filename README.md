# Collider Blocker

## 設定項目
#### コライダーを検出
アバターのコライダーを検出します。
> [!WARNING]
> サイズが0のコライダーは検出できません。
#### コライダーダッシュを禁止
プレイヤーの速度を監視し、速い場合は遅くします。
#### 許容速度
Blockerが許容する速度です。誤検知を防ぐため、若干の速度超過を許容することをお勧めします。
### コライダー検出時の動作
#### ジャンプを無効化
ジャンプを無効化します。これでコライダージャンプの発動を阻止できるはずです。
#### プレイヤーを固定
プレイヤーをその場に固定します。リスポーンは可能です。
#### 検出時に表示するGameObject
Blocker発動を知らせるHUDなどを指定してください。混乱を防ぐため、導入を推奨します。
> [!IMPORTANT]
> コライダーダッシュの発動時、これは表示されません。
